# Whatsapper — How To Get Your Key (5 steps, free)

1. Go to **developers.facebook.com** → **Create app** → add the **WhatsApp** product.
2. Open **WhatsApp → API Setup** → copy your **Phone number ID**.
3. Click **Generate access token** → copy the **token** (keep it secret).
4. Paste both into Tlamatini's `config.json` (`whatsapp_phone_number_id`, `whatsapp_access_token`) → restart.
5. Your first message must use a template (try `hello_world`); once someone replies, you can send normal text.

Need every click with screenshots? See **StepByStepProcessUsingWhatsapper.md**.
Want the *easy* messenger instead? **Telegramer** needs only a bot token from **@BotFather** (~2 minutes).
