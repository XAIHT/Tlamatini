# ── tkinter / Tcl-Tk : NOT used by the Tlamatini SERVER ──────────────────
# The server's Set-DB / Backup-DB "Browse" buttons use the Win32 common
# dialogs directly via ctypes (agent/native_dialogs.py -> comdlg32/shell32,
# which ship with every Windows install). The server therefore has NO
# tkinter / Tcl-Tk dependency — build.py excludes it (--exclude-module),
# so there is nothing to bundle and the old "Can't find a usable init.tcl"
# failure can no longer occur.
#
# tkinter IS still used by the standalone Installer/Uninstaller GUIs
# (install.py / uninstall.py), which are SEPARATE executables built by
# build_installer.py / build_uninstaller.py — those scripts locate and
# bundle their own Tcl/Tk (they set TCL_LIBRARY/TK_LIBRARY at build time).
# tkinter is CPython stdlib (the "tcl/tk and IDLE" optional component) and
# is not pip-installable, so it is intentionally absent from this file.
